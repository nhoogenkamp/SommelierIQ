SommelierIQ

SommelierIQ is a web-based wine recommendation system developed for F.X. Buckley Steakhouses. The application helps customers and restaurant staff identify suitable wines based on selected dishes, sauces, and budget preferences. It also provides an administrative dashboard for managing wine records through secure CRUD operations.
